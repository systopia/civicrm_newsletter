Basic Configuration
* Install Module
* Add, edit or check CMR Profile (/admin/config/cmrf_profile)
* Add CMRF Connector(/admin/config/cmrf_connector; Systemname: civicrm_newsletter, Connecting Module: civicrm_newsletter; Profile: default)
* Select defined CMRF Connector (/admin/config/services/civicrm_newsletter)

You should then be able to access your default subscription page: /civicrm_newsletter/subscribe/default
